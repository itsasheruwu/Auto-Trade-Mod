## Auto Trade Mod v1.0.4

### Features
- Added `Keep auto ON when trading UI closes` setting.
- Auto mode can now stay armed across villager UI close/reopen and resume automatically.

### Updater
- Auto-updater now checks during startup and pre-launch.
- New releases are downloaded and installed earlier in the launch flow.
- Clarified behavior: updated code still requires restarting Minecraft.

### Stability
- Improved startup updater flow with one-time check guard and safer error handling.
- Improved UI status behavior when auto mode remains armed between screens.
